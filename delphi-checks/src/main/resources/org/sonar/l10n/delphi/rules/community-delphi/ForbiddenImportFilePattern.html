Why is this an issue?

This rule allows banning certain imports. It will flag any import of a unit with a file path matching the forbiddenImportPattern.

Please note:

  • Patterns are compared against the unit's absolute path.
  • Patterns can use glob or regex syntax, depending on the value of forbiddenImportPatternSyntax and whitelistPatternSyntax.
  • Units with a file path matching the optional whitelistPattern are permitted to import banned units.
